Introduction to the GrAL

GrAL is the Grid Algorithms Library. It contains generic algorithms and data structures working on grids (or meshes), such as used in finite element modelling, but also in computer graphics, geometric modelling and so on. It also contains some concrete grid data structures, but the focus is clearly on the generic (data-structure neutral) stuff.

GrAL consist of
  • a growing number (currently 4) of grid-related modules,
  • some low-level support modules, which are not part of GrAL in a proper sense, but are used at some places.
GrAL is very much work in progress. The contents (especially grid-related modules) are not very complete at the moment, and are expected to grow quite a bit.
For an in-depth treatment of the concepts underlying GrAL, see Guntram Berti's PhD dissertation .
